# calcValidConsumptionValue

Validation for consumption Value

## Usage

``` r
calcValidConsumptionValue(datasource = "FAO")
```

## Arguments

- datasource:

  datasource for validation (FAO, FAOpre2010, FAOpost2010)

## Value

List of magpie object with results on country level, no weight, unit and
description.
