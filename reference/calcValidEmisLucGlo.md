# calcValidEmisLucGlo

Returns historical and projected water usage from different sources

## Usage

``` r
calcValidEmisLucGlo(subtype = "Canadell_2007")
```

## Arguments

- subtype:

  Available subtypes are:

  - historical:

    - Canadell_2007

    - Friedlingstein_2010

    - Harris_2013

    - Houghton_2012

    - RCP

## Value

list of magpie object with data and weight
