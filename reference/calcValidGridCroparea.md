# calcValidGridCroparea

reports Croparea by crops on 0.5 degree grid in physical area

## Usage

``` r
calcValidGridCroparea(physical = TRUE)
```

## Arguments

- physical:

  if true (default) physical area; if false harvested area

## Value

List of magpie objects with results on cellular level, weight on
cellular level, unit and description.
