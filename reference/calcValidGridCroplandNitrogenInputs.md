# calcValidGridCroplandNitrogenInputs

reports Nitrogen Inputs by crop type on 0.5 degree grid

## Usage

``` r
calcValidGridCroplandNitrogenInputs()
```

## Value

List of magpie objects with results on cellular level, weight on
cellular level, unit and description.
