# readAR6SPMFig1

download IPCC AR6 Summary for Policymakers Fig 1, global surface
temperature change relative to 1850-1900 from observations. This dataset
also includes the CMIP6 models (human and natural forcings simulations),
but are so far excluded here.

## Usage

``` r
readAR6SPMFig1()
```

## Value

magclass object

## See also

https://www.ipcc.ch/report/ar6/wg1/downloads/report/IPCC_AR6_WGI_SPM.pdf
