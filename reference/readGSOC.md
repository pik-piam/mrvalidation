# readGSOC

This function reads the raw GSOC data (available at
http://54.229.242.119/GSOCmap/) or if available the preprocessed raster
layers.

## Usage

``` r
readGSOC()
```

## Value

magpie object in cellular resolution

## Examples

``` r
if (FALSE) { # \dontrun{
readSource("GSOC")
} # }
```
