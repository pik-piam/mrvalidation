# readGlobalCarbonBudget

read the Global Carbon Budget, selecting the models GCB, BLUE, H&C2023,
OSCAR and their sub-components Net, Deforestation, Forest regrowth,
Other transitions, Wood harvest and other forest management

## Usage

``` r
readGlobalCarbonBudget()
```

## Value

a magpie object in Mt CO2 per year

## Author

Michael Crawford

## Examples

``` r
if (FALSE) { # \dontrun{
readSource("GlobalCarbonBudget")
} # }
```
