# correctSoilCarbonDebt

Correct data from Soil Carbon Debt Paper
(https://github.com/whrc/Soil-Carbon-Debt/)

## Usage

``` r
correctSoilCarbonDebt(x)
```

## Arguments

- x:

  magpie object provided by the read function

## Value

List of magpie objects

``` r
if (FALSE) { # \dontrun{
```
