# FAOrotation

Read Rotation lengths reported by Global planted forests thematic study:
Results and analysis (2006)

## Usage

``` r
readFAOrotation(subtype = "FAO")
```

## Arguments

- subtype:

  subtype - only FAO supported

## Value

magpie object containing rotation length data

## See also

[`readSource`](https://rdrr.io/pkg/madrat/man/readSource.html)
