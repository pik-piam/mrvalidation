# calcValidTimber

Returns historical timber demand and production from FAO in volumetric
units (Mm3/yr). Variable names match reportTimber.

## Usage

``` r
calcValidTimber(datasource = "FAO")
```

## Arguments

- datasource:

  Currently only available for the "FAO" source

## Value

List of magpie object with data and population
