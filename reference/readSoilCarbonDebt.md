# readSoilCarbonDebt

Read data from Soil Carbon Debt Paper
(https://github.com/whrc/Soil-Carbon-Debt/)

## Usage

``` r
readSoilCarbonDebt()
```

## Value

List of magpie objects with results on cellular level, weight, unit and
description.

``` r
if (FALSE) { # \dontrun{
readSource("SoilCarbonDebt", subtype = "SOCS_2010")
} # }
```
