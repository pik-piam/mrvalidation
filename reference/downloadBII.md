# downloadBII

Download historical and projected BII dataset from Phillips et al. 2021
(https://doi.org/10.5519/he1eqmg1).

## Usage

``` r
downloadBII()
```

## Value

Metadata on downloaded BII data

## See also

https://data.nhm.ac.uk/dataset/bii-bte
