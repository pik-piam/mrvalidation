# calcValidGridNitrogenBudgetPasture

reports Nitrogen Budgets for Pastures on 0.5 degree grid

## Usage

``` r
calcValidGridNitrogenBudgetPasture()
```

## Value

List of magpie objects with results on cellular level, weight on
cellular level, unit and description.
