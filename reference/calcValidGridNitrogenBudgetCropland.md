# calcValidGridNitrogenBudgetCropland

reports Nitrogen Budget for Croplands on 0.5 degree grid

## Usage

``` r
calcValidGridNitrogenBudgetCropland()
```

## Value

List of magpie objects with results on cellular level, weight on
cellular level, unit and description.
