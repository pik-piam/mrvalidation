# calcValidCostsOverall

Returns historical values of the overall value of production for the
Agricultural, Forestry sectors, and fisheries (mio.17USD)

## Usage

``` r
calcValidCostsOverall(datasource = "FAO")
```

## Arguments

- datasource:

  datasource for validation. FAO Value of Production dataset.

## Value

List of magpie object with results on country level, no weight, unit and
description.
