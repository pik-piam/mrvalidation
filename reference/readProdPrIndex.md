# Read FAO producer price index

Read-in world producer price index as magclass object

## Usage

``` r
readProdPrIndex()
```

## Value

magpie object of the world producer index from FAO

## See also

[`readSource`](https://rdrr.io/pkg/madrat/man/readSource.html)

Mishko Stevanovic

## Examples

``` r
if (FALSE)  a <- readSource(type="ProdPrIndex")
 # \dontrun{}
```
