# calcValidPriceGHG

Validates MAgPIE GHG emission price input against SSP GHG emission price
projections

## Usage

``` r
calcValidPriceGHG(datasource = "SSPResults")
```

## Arguments

- datasource:

  Currently available: `"SSPResults"`
