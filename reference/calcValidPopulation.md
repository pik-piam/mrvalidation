# calcValidPopulation

Returns historical development of population and future projections of
population dynamics

## Usage

``` r
calcValidPopulation(
  pastData = "WDI-UN_PopDiv-MI",
  futureData = "SSPs-UN_PopDiv"
)
```

## Arguments

- pastData:

  population past data source

- futureData:

  population future data source

## Value

list of magpie object with data and weight
