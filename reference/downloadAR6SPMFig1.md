# downloadAR6SPMFig1

download IPCC AR6 Summary for Policymakers Figure 1, global surface
temperature change relative to 1850-1900 from observations and CMIP6
models (human and natural forcings simulations)

## Usage

``` r
downloadAR6SPMFig1()
```

## Value

Metadata on downloaded dataset

## See also

https://www.ipcc.ch/report/ar6/wg1/downloads/report/IPCC_AR6_WGI_SPM.pdf
