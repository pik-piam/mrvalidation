# calcValidAEI

Returns historical area equipped for irrigation.

## Usage

``` r
calcValidAEI(datasource = "LUH3")
```

## Arguments

- datasource:

  Currently available: `"LUH3"`, `"HID"`, `"GMIA"`, and `"Mehta2022"`

## Value

list of magpie object with data and weight
