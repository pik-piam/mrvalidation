# calcValidEmisLucGasser

Returns historical LUC emissions

## Usage

``` r
calcValidEmisLucGasser(subtype = "bookkeeping")
```

## Arguments

- subtype:

  Available subtypes are:

  - historical:

    - Gasser_2020

    - LUH2_GCB_2019

    - FRA_2015

    - bookkeeping

## Value

list of magpie object with data and weight
