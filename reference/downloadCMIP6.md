# downloadCMIP6

Download near-surface air temperature projection from CMIP6 for the
following scenarios: SSP1-1.9, SSP1-2.6, SSP2-4.5, SSP3-7.0, SSP4-3.4,
SSP4-6.0, SSP5-8.5, SSP5-3.4-over

## Usage

``` r
downloadCMIP6()
```

## Value

Metadata on downloaded CMIP near-surface air temperature data

https://cmip6.science.unimelb.edu.au/
