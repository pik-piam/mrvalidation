# calcValidEmisLucGloGasser

Returns historical LUC emissions

## Usage

``` r
calcValidEmisLucGloGasser(subtype = "Gasser_2020")
```

## Arguments

- subtype:

  Available subtypes are:

  - historical:

    - Gasser_2020

    - LUH2_GCB_2019

    - FRA_2015

## Value

list of magpie object with data and weight
