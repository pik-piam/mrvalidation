# calcValidAgFFGDP

validation for agricultural, fisheries, forestry value added gdp
(Million 17USD)

## Usage

``` r
calcValidAgFFGDP(datasource = "WDI")
```

## Arguments

- datasource:

  datasource for validation (WDI)

## Value

List of magpie object with results on country level, no weight, unit and
description.
