# calcValidCostsAEI

provides the validation data for irrigation investment costs

## Usage

``` r
calcValidCostsAEI(datasource = "IMPACT")
```

## Arguments

- datasource:

  Datasource of validation data

## Value

magpie object on country level, unit and description
