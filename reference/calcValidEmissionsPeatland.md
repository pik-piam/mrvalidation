# calcValidEmissionsPeatland

Validation data for peatland emissions in CO2eq, taken from the article:
Leifeld, J., Menichetti, L. The underappreciated potential of peatlands
in global climate change mitigation strategies. Nat Commun 9, 1071
(2018). https://doi.org/10.1038/s41467-018-03406-6

## Usage

``` r
calcValidEmissionsPeatland()
```

## Value

Global-level MAgPIE object with peatlands emissions in CO2eq for the
year 2015
